This directory contains, in the two sub-directories, the files 
representing various chemical structures for testing InChI/InChIKey software. 


=========
  FILES 
=========

readme.txt                       This file

samples				 SUB-DIRECTORY

Sample chemical structure files for testing InChI/InChIKey software.  

test				 SUB-DIRECTORY

Test suite including data file (SDF) and reference InChI/InChIKey identifiers 
obtained with InChI software v. 1.03 using various command-line options.
